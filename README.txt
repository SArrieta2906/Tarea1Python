Este programa genera una imagen en formato png de acuerdo con las instrucciones dadas en codigo.txt, en caso de encontrar errores en las instrucciones de codigo.txt 
se generará un errores.txt indicando las instrucciones con su respectiva ubicación que ocasionan el mal funcionamiento del programa.

Un código de ejemplo con las instrucciones necesarias se encuentra en codigo.txt. Para ejecutar el programa se debe ejecutar el archivo pixelart.py
